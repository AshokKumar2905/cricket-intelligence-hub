# 🏏 Cricket Intelligence Hub (DevOps Edition)

A high-performance, containerized analytics engine for professional cricket data tracking.

## 🚀 DevOps Architecture
This project demonstrates a production-ready microservices orchestration:
- **Unified Gateway**: React (Vite) + Nginx SPA Routing.
- **Analytics Engine**: Python Flask with Pandas data processing.
- **Data Node**: MongoDB with persistent Docker Volumes.
- **Orchestration**: Docker Compose with automated Health Check protocols.

## 🛠️ Rapid Setup
1. Clone the repository.
2. Ensure Docker Desktop is active.
